Any questions?

How does this extension work?
Yes! SoundLine is an open source project that supports a large number of people. You can get acquainted with all the internal functionality and all the code by clicking on this link.
How much does it cost?
To get acquainted, you can go to the prices page (ссылка на /pricing) and choose the right tariff for yourself. Please note that you can use the SoundLine for free, but only in limited functionality.
How do I use it to record calls on Zoom or Microsoft Teams?
To use the extension about recording calls and using the AI translator SoundLine, you need to synchronize the application with the site, go to the "plugins" and select the appropriate section, synchronizing the desired application with your AI assistant.
Do I need to download an app to make a call?
Yes, for the first connection, you need to synchronize the SoundLine working application with the web version of the site. In the future, this will need to be done only when updating the functionality from the technical part. You can use SoundLine directly from the browser.
How is the sound recorded?
We record the audio as you hear it when you are in the call. This means the sound will properly be recorded, even if you wear headphones.
Will a "bot" join my meeting to record the call, like other tools?
No. It will only be you and the people with whom you have planned a call with.
